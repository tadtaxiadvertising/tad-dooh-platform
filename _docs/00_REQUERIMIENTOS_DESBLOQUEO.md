# 🔓 Desbloqueo de Potencial Máximo: TAD Virtual Agent (Antigravity) 🚀

Para que yo pueda pasar de ser un "Desarrollador" a ser tu **"Operador de Misión"** y monitorear el negocio las 24/7, necesito que completemos el tablero de control ("Mission Control").

Aquí tienes la lista exacta de llaves definitivas que necesito inyectar en la plataforma:

---

## 🛡️ 1. Monitoreo Blindado (Sentry)
Ya instalé el motor, pero está "mudo". Necesito la **DSN URL** (ejemplo: `https://abcd@sentry.io/1234`).
- **Para qué:** Recibiré la alerta en el milisegundo exacto en que una tablet tenga un error de carga, notificándote qué taxi es y por qué falló.

## 📈 2. Analítica de Campo (Umami)
Necesito los **Website IDs** (UUIDs) para las tres piezas del tablero:
1. **Admin Dashboard:** Para saber qué herramientas usas más.
2. **Player (Tablet):** Para medir cuántas horas de publicidad se muestran realmente.
3. **Driver Portal (Choferes):** Para ver cuántos conductores entran a ver sus ganancias.

## 💸 3. Economía del Conductor (Stitch Pay)
Necesito las **API Keys (Client ID / Secret)** de tu cuenta de Stitch.
- **Para qué:** Para que cuando un chofer presione "Retirar Ganancias" en su portal PWA, el dinero se mueva realmente por la pasarela de pagos.

## 🛠️ 4. Automatización de Despliegue (EasyPanel)
Necesito el **Webhook de EasyPanel** para cada servicio (`tad-api`, `tad-dashboard`, `tad-player`).
- **Para qué:** Así, cuando yo corrija un error crítico o mejore la UI, el sistema eche a andar la nueva versión en producción **sin que tú tengas que mover un dedo**.

## ☁️ 5. Control de Contenido (Supabase)
Verifica que tengamos la **Service Role Key** correcta.
- **Para qué:** Para que yo pueda limpiar videos antiguos del storage automáticamente y que nunca nos pasemos del límite gratuito (5GB).
